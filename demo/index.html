Get started

Use the snippet below as an example how to get started. The idea is to easily create an instance with options and you're all set. All other functionality is set and used via data-attribute tags. 



$(document).ready(function() {

  // Create instance with options.
  $(window).gamepad({
    message: "what"
  });

});
